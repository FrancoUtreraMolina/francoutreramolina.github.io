Proyecto

El sentido de este proyecto es el Demostrar nuestro aprendizajes de las clases de CPT.

De Manera que por medio de esta pagina explicare las partes que la componen

Style

Para empezar pusimos una etiqueta Style dentro del apartado de Head, la etiqueta Style la podemos utilizar para dar color y diseño al Header, Nav y Footer

Header

Luego integramos la etiqueta Header dentro de la seccion de Body, esta etiqueta nos sirve para integra un titulo como encabezado.

Nav

Tambien integramos la etiqueta Nav, la cual la podemos visualisar en la pagina en el costado izquierdo, esta al igual que las demas se encuentra dentro de body.

Para terminar integramos la etiqueta Footer la cual la podemos encontrar en la parte baja de la pagina, y luego de ella cerramos la etiqueta body y cerrando el html finalizando con nuestra pagina.
